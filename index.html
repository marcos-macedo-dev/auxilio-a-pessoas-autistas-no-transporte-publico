Auxílio a Pessoas Autistas no Transporte Público

Projeto Integrador · IoT e Cidades Inteligentes · UniFAP 2025

Abrir documento principal (PDF)

Enfoque

Transporte público costuma ser caótico para pessoas no espectro. A proposta entrega previsibilidade em rotas de ônibus para reduzir sobrecarga sensorial.

  • Alinhado à ODS 11.2 (mobilidade acessível).
  • Uso mínimo de hardware para máximo impacto.
  • Informação clara e antecipada em pontos-chave do trajeto.

Componentes

Ônibus

ESP32/Arduino com LoRa e painel LED/LCD exibindo próxima parada, distância e tempo estimado.

Parada

Gateway com LoRa + 4G que recebe dados dos ônibus e mostra lista de veículos, distâncias e tempos.

Servidor

Node/Express com MQTT (HiveMQ) e Socket.IO; guarda telemetria em JSON/banco e distribui ao dashboard.

Dashboard

Interface web para monitorar veículo, rota, próximas paradas e histórico recente.

Fluxo de dados

Sequência

  • Ônibus → parada via LoRa.
  • Parada → servidor via 4G.
  • Servidor → displays e web via MQTT/Socket.IO.

Protocolo

LoRa/LoRaWAN (classe C) para pacotes curtos; MQTT para telemetria; Socket.IO para tempo real na web.

Dados e persistência

Campos principais

Veículo (id, rota), parada atual, próxima parada, distância (km), tempo estimado (min), timestamp.

Histórico

Eventos de mudança de parada guardados para consulta e exibição em tempo real.

Impacto

Inclusão

Antecipação reduz estresse sensorial e dá segurança a passageiros autistas.

Escala

Componentes baratos e arquitetura simples favorecem expansão para várias linhas.

ODS

Contribui para transporte urbano acessível, seguro e previsível.
